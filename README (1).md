# 🎉 DevWebCamp

Aplicación web desarrollada para la gestión integral de eventos, conferencias y registro de participantes.

## 🚀 Características

- Registro e inicio de sesión de usuarios.
- Gestión de eventos y conferencias.
- Administración de ponentes.
- Registro de asistentes.
- Compra y gestión de boletos.
- Panel de administración.
- Diseño responsive.

## 🛠️ Tecnologías utilizadas

- PHP 8+
- MVC
- Composer
- HTML5
- SCSS
- JavaScript
- TypeScript
- SQL Server
- Git y GitHub

## ⚙️ Instalación

```bash
git clone https://github.com/Yasuo05/devwebcampOP.git
cd devwebcampOP
composer install
```

Importa la base de datos `BD.sql` y configura las credenciales de conexión.

Para ejecutar:

```bash
php -S localhost:8000
```

Abre:

`http://localhost:8000`

## 📌 Funcionalidades

- Autenticación
- Gestión de eventos
- Gestión de conferencias
- Gestión de participantes
- Venta de boletos
- Panel administrativo
